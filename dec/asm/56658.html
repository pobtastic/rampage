56658: Sets IY To Monster Data
Used by the routines at Handler_Eating, Handler_EatFood and Handler_Monsters.
Input
A Monster ID
Output
IY Monster data
SetIYMonsterData 56658 CP 1 Jump to SetIYMonsterData_Lizzy if A is not equal to 1.
56660 JR NZ,SetIYMonsterData_Lizzy
A is 1; set to George:
56662 LD IY,53202 IY=George_State.
56666 RET Return.
If it's not 1 is it 2?
SetIYMonsterData_Lizzy 56667 CP 2 Jump to SetIYMonsterData_Ralph if A is not equal to 2.
56669 JR NZ,SetIYMonsterData_Ralph
A is 2; set to Lizzy:
56671 LD IY,53249 IY=Lizzy_State.
56675 RET Return.
Just return if A is none of 1 2 or 3.
SetIYMonsterData_Ralph 56676 CP 3 Return if A is not equal to 3.
56678 RET NZ
A is 3; set to Ralph:
56679 LD IY,53296 IY=Ralph_State.
56683 RET Return.
